How to open on your iPhone
Host the app free on GitHub Pages, then install it to your iPhone home screen like a native app.
1
Go to github.com, create a free account, then click New repository. Name it shredmtb and set it to Public.
2
Click uploading an existing file and drag in all four files: index.html, manifest.json, sw.js, icon.svg. Commit.
3
Go to Settings → Pages, set Source to Deploy from a branch → main → / (root) and save.
4
Your app will be live at:
https://YOUR-USERNAME.github.io/shredmtb/
5
Open that URL on your iPhone in Safari → tap the Share button → Add to Home Screen. Done — it installs like a native app, works offline, and GPS tracking works on trail.
